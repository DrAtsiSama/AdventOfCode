# Advent of Code 2025 – Go

Repo perso pour Advent of Code 2025 en Go.

## Structure

- `dayXX/`
  - `main.go` : solution du jour (part 1 & part 2)
  - `input.txt` : input officiel du jour
  - `prompt.md` : énoncé ou notes perso

Exemple :

```text
AOC2025/
├─ go.mod
├─ README.md
├─ day01/
│  ├─ main.go
│  ├─ input.txt
│  └─ prompt.md
└─ day02/
   ├─ main.go
   ├─ input.txt
   └─ prompt.md
